Política de Privacidade — COTW Wild Life

Última atualização: 30/11/2025

O aplicativo COTW Wild Life respeita a privacidade do usuário e se compromete a garantir que nenhuma informação pessoal seja coletada, utilizada ou compartilhada.

1. Coleta de Dados

O aplicativo não coleta, armazena, transmite ou compartilha quaisquer dados pessoais dos usuários. Isso inclui, mas não se limita a:

  • Nome, e-mail ou telefone
  • Dados de login
  • Localização
  • Arquivos pessoais
  • Informações de dispositivo
  • Identificadores únicos
  • Dados sensíveis

Nenhum dado é enviado para servidores externos ou armazenado em nuvem.

2. Permissões

O aplicativo não solicita permissões especiais, além das necessárias para seu funcionamento padrão. O app não acessa:

  • Internet
  • Localização
  • Microfone
  • Arquivos pessoais
  • Contatos
  • Calendário

3. Armazenamento Local

Algumas informações podem ser salvas localmente no dispositivo, como:

  • Preferências de idioma
  • Configurações do aplicativo

Esses dados nunca deixam o dispositivo do usuário.

4. Crianças

O aplicativo é adequado para todas as idades. Como não há coleta de dados, o app está em conformidade com a LGPD (Brasil), GDPR (Europa) e diretrizes gerais para proteção infantil.

5. Compartilhamento de Dados

Nenhuma informação, de qualquer tipo, é compartilhada com terceiros.

6. Segurança

Como o COTW Wild Life não coleta nem transmite dados pessoais, não há risco de exposição ou vazamento de informações do usuário.

7. Alterações na Política

Esta política pode ser atualizada para refletir melhorias no aplicativo. Sempre que isso ocorrer, a versão mais recente estará disponível nesta página.
